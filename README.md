# Installation steps
***
1. Install python 3.11 (recommended) or later versions
2. Install pip 
3. Install dependencies using pip install **
    . plotly
    . dash
4. update config variables (mv config.example.py config.py)
5. load data
    python3.11 data_storage.py (update file path for data storage)
6. run python3.11 app.py to visualize data
    default port is 8050 (localhost:8050)
***
